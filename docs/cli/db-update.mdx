Ensure your diagnostics are accurate by keeping your local database up to date.

## Usage

```bash
prism db update
```

## What it updates
- **Error Categories**: Maps for new host errors introduced in protocol upgrades.
- **Suggested Fixes**: Community-contributed solutions for common failures.
- **WASM Spec Mappings**: Logic for parsing latest `contractspecv0` versions.
