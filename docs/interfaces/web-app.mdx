The Prism Web App provides a visual interface for analyzing Soroban transactions.

## Features

- **Paste-and-Diagnose**: Simply paste a transaction hash and select the network to start.
- **Interactive Trace Viewer**: Collaspible invocation trees and visual budget consumption charts.
- **Shareable URLs**: Every debug session gets a unique URL — no login required to share a trace with your team.
- **Web-Based Replay**: Step through transactions in the browser using the WASM-powered engine.
